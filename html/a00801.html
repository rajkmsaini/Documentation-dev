Detailed Description

template<class LimiterFunc>
class Foam::filteredLinear3Limiter< LimiterFunc >

Class to generate weighting factors for the filteredLinear differencing scheme.

The aim is to remove high-frequency modes with "staggering" characteristics by comparing the face gradient with both neighbouring cell gradients and introduce small amounts of upwind in order to damp these modes.

Used in conjunction with the template class LimitedScheme.
