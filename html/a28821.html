Detailed Description

template<class Type>
class Foam::Function1Types::Sine< Type >

Templated sine function with support for an offset level.

where

= Description Data type
= Amplitude Function1<scalar>
= Frequency [1/s] Function1<scalar>
= Type scale factor Function1<Type>
= Type offset level Function1<Type>
= Start time [s] scalar
= Time [s] scalar

Example for a scalar: 

        <entryName> sine;
        <entryName>Coeffs
        {
            frequency 10;
            amplitude 0.1;
            scale     2e-6;
            level     2e-6;
        }

Example for a vector: 

        <entryName> sine;
        <entryName>Coeffs
        {
            frequency 10;
            amplitude 1;
            scale     (1 0.1 0);
            level     (10 1 0);
        }
