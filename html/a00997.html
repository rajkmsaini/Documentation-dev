Detailed Description

This class provides help for boundary conditions (patch fields). When no additional arguments are given, the utility outputs all known boundary conditions.

Usage:

-browse <patch type> : open documentation for patch in browser
-field <name> : boundary conditions for a given field
-constraint : list constraint boundary conditions
-fixedValue : list fixedValue type boundary conditions
Note
To use the -fixedValue option, the -field option should also be used
