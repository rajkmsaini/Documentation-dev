25  Foam::writeRegisteredObject
26 
27 Group
28  grpIOFunctionObjects

30 Description
31  This function object allows specification of different writing frequency
32  of objects registered to the database. It has similar functionality
33  as the main time database through the outputControl setting:
34  timeStep
35  outputTime
36  adjustableTime
37  runTime
38  clockTime
39  cpuTime
40 
41  Example of function object specification:
42  \verbatim
43  writeRegisteredObject1
44  {
45  type writeRegisteredObject;
46  functionObjectLibs ("libIOFunctionObjects.so");
47  exclusiveWriting true;
48  ...
49  objectNames (obj1 obj2);
50  }
51  \endverbatim
52 
53  \heading Function object usage
54  \table
55  Property | Description | Required | Default value
56  type | type name: writeRegisteredObject | yes |
57  objectNames | objects to write | yes |
58  exclusiveWriting | Takes over object writing | no | yes
59  \endtable
60 
61  exclusiveWriting disables automatic writing (i.e through database) of the
62  objects to avoid duplicate writing.
