95 //- Search for files from user/group/shipped directories.
96 // The search scheme allows for version-specific and
97 // version-independent files using the following hierarchy:
98 // - \b user settings:
99 // - ~/.OpenFOAM/<VERSION>
100 // - ~/.OpenFOAM/
101 // - \b group (site) settings (when $WM_PROJECT_SITE is set):
102 // - $WM_PROJECT_SITE/<VERSION>
103 // - $WM_PROJECT_SITE
104 // - \b group (site) settings (when $WM_PROJECT_SITE is not set):
105 // - $WM_PROJECT_INST_DIR/site/<VERSION>
106 // - $WM_PROJECT_INST_DIR/site/
107 // - \b other (shipped) settings:
108 // - $WM_PROJECT_DIR/etc/
109 //
110 // \return The list of full paths of all the matching files or
111 // an empty list if the name cannot be found.
112 // Optionally abort if the file cannot be found.
113 // Optionally stop search after the first file has been found.
