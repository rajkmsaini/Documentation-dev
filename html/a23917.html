Detailed Description

template<class LimiterFunc>
class Foam::vanAlbadaLimiter< LimiterFunc >

Class with limiter function which returns the limiter for the vanAlbada differencing scheme based on r obtained from the LimiterFunc class.

Used in conjunction with the template class LimitedScheme.
