Detailed Description

Fixed temperature equation constraint.

Sources described by:

fixedTemperatureConstraintCoeffs { mode uniform; // uniform or lookup

uniform option temperature constant 500; // fixed temperature with time [K]

lookup option TName T; // optional temperature field name }

Note: The 'uniform' option allows the use of a time-varying uniform temperature by means of the DataEntry type.
