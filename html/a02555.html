Detailed Description

template<class LimiterFunc>
class Foam::SuperBeeLimiter< LimiterFunc >

Class with limiter function which returns the limiter for the SuperBee differencing scheme based on r obtained from the LimiterFunc class.

Used in conjunction with the template class LimitedScheme.
