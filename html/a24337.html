Detailed Description

Calculates and write the heat-flux at wall patches as the volScalarField field 'wallHeatFlux'.

All wall patches are included by default; to restrict the calculation to certain patches, use the optional 'patches' entry.

Example of function object specification: 

wallHeatFlux1
{
    type        wallHeatFlux;
    libs        ("libfieldFunctionObjects.so");
    ...
    patches     (".*Wall");
}
Usage
Property Description Required Default value
type type name: wallHeatFlux yes
patches list of patches to process no all wall patches
