Detailed Description

This functionObject writes objects registered to the database in VTK format using the foamToVTK library.

Currently only the writing of the cell-values of volFields is supported but support for other field types, patch fields, Lagrangian data etc. will be added.

Example of function object specification: 

    writeVTK1
    {
        type        writeVTK;
        functionObjectLibs ("libIOFunctionObjects.so");
        ...
        objectNames (obj1 obj2);
    }
