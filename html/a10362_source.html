25  Foam::radiation::viewFactor
26 
27 Description
28  View factor radiation model. The system solved is: C q = b
29  where:
30  Cij = deltaij/Ej - (1/Ej - 1)Fij
31  q = heat flux
32  b = A eb - Ho
33  and:
34  eb = sigma*T^4
35  Ej = emissivity
36  Aij = deltaij - Fij
37  Fij = view factor matrix
