Detailed Description

Solid-body 6-DoF acceleration source.


Source usage

Example usage: 

SBM
{
    type            tabulatedAccelerationSource;
    active          yes;

    tabulatedAccelerationSourceCoeffs
    {
        timeDataFileName "constant/acceleration-terms.dat";
    }
}
