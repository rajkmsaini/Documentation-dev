25  Foam::coordinateSystem

27 Description
28  Base class for other coordinate system specifications.
29 
30  All systems are defined by an origin point and a co-ordinate rotation.
31 
32  \verbatim
33  coordinateSystem
34  {
35  type cartesian;
36  origin (0 0 0);
37  coordinateRotation
38  {
39  type cylindrical;
40  e3 (0 0 1);
41  }
42  }
43  \endverbatim
44 
45  Types of coordinateRotation:
46 
47  \li 1. axesRotation
48  \li 2. STARCDRotation
49  \li 3. cylindrical
50  \li 4. EulerCoordinateRotation
51 
52  Type of co-ordinates:
53 
54  \li 1. cartesian
