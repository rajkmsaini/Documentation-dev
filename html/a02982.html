Detailed Description

Calculates and write the shear-stress at wall patches as the volVectorField field 'wallShearStress'.

where

= stress tensor
= patch normal vector (into the domain)

The shear-stress symmetric tensor field is retrieved from the turbulence model. All wall patches are included by default; to restrict the calculation to certain patches, use the optional 'patches' entry.

Example of function object specification: 

    wallShearStress1
    {
        type        wallShearStress;
        libs        ("libfieldFunctionObjects.so");
        ...
        patches     (".*Wall");
    }
Usage
Property Description Required Default value
type type name: wallShearStress yes
patches list of patches to process no all wall patches
