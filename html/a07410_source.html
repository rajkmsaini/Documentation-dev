25  Foam::cylindrical
26 
27 Description
28  A local coordinate rotation.
29  The cell based rotational field can be created in two ways:
30 
31  \li 1. Each rotational tensor is defined with two vectors (\c dir and \c e3)
32  where <tt>dir = cellC - origin</tt> and \c e3 is the rotation axis.
33  Per each cell an axesRotation type of rotation is created
34  (cylindrical coordinates). For example:
35  \verbatim
36  cylindrical
37  {
38  type localAxes;
39  e3 (0 0 1);
40  }
41  \endverbatim
42 
43  \li 2. The rotational tensor field is provided at construction.
